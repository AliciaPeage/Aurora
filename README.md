# FlixBot
Official "KvasirSG Development" Discord Bot

http://kvasirsg.com/

## Requirements

1. Python 3
2. discord.py python package `pip install discord.py`

## Install and run

1. Extract everything from the archive
2. Rename `rm_auth.py` in `config` to `auth.py`
3. Configure the bot in `auth.py`
